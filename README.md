# Google AI Overview Remover

## 📌 Overview

**Google AI Overview Remover** is a lightweight Chrome extension designed to restore clarity to your search experience by dynamically removing **Google AI Overviews (Gemini-generated answers)** from the top of your search results.

It identifies and intercepts AI-generated blocks in real-time as the page loads, ensuring your search results remain fast, clean, and distraction-free.

## ⚙️ Features

* 🚫 **Instant Removal:** Strips AI Overview sections from Google Search results.
* ⚡ **Real-Time Detection:** Uses a `MutationObserver` to catch dynamically injected content.
* 🧩 **Safe & Lightweight:** No impact on browser performance or other Google UI features.
* 🧠 **Advanced Heuristics:** Detects AI blocks using CSS classes, attributes, and text patterns.
* 🛠️ **Debug Mode:** Includes a specialized script for DOM pattern analysis.

## 🚀 Installation (Load Unpacked)

1. Open **Chrome** or any Chromium-based browser.
2. Navigate to `chrome://extensions/`.
3. Enable **Developer Mode** (toggle in the top-right).
4. Click **Load Unpacked** and select this project’s folder.
5. Ensure the extension is **enabled**.

## 🔍 Detection Logic

**Primary signals (`isAIOverview(element)`):**
* Specific Google UI classes: `.Fzsovc`, `.YzCcne`, `.g-blk`, `.MjjYud`
* Hidden headings and labels (e.g., *AI Overview*, *Generated by Gemini*)
* Attributes like `data-mcpr`, `data-mg-cp`, and `jscontroller`

**Strategy:**
The extension scans known selectors and removes the nearest parent container while monitoring the DOM for new insertions. Built-in safeguards (`isCriticalElement`) prevent accidental removal of essential search tools.
