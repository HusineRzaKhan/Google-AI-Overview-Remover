## ЁЯФН Detection Logic

**Primary detection signals (`isAIOverview(element)`):**

* Hidden or clipped headings (e.g., *AI Overview*, *Generated by AI*)
* Known Google classes: `.Fzsovc`, `.YzCcne`, `.g-blk`, `.MjjYud`
* Attributes like `data-mcpr`, `data-mg-cp`, `data-ai`, `jscontroller`
* Text markers: тАЬAI OverviewтАЭ, тАЬAI-generatedтАЭ, тАЬGenerated by GeminiтАЭ

**Removal Strategy:**

1. Scan known selectors and remove nearest parent container.
2. Monitor `.g`, `.tF2Cxc`, `.MjjYud`, `.g-blk` containers for nested AI content.
3. Perform fallback limited scans to ensure missed AI blocks are removed.
