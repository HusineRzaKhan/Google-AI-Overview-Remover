Test Instructions

  1. Load this extension in Chrome
  2. Go to Google Search with AI Overview
  3. Check if AI Overview sections are removed
  4. Verify that regular search results are still visible

What the Extension Should Remove

The extension should remove elements that contain:

  • Text containing "AI Overview", "AI-generated", "Gemini", etc.
  • Elements with AI-related attributes like data-ai, data-ved*="ai"
  • Elements with classes like ai-overview, gemini

Extension Status

Check the browser console (F12) for extension messages like:

Google AI Overview Remover: Observer initialized successfully
Google AI Overview Remover: Removed X AI Overview elements

Troubleshooting

  • Page goes blank: The extension might be too aggressive. Check console for errors.
  • AI Overview not removed: Google might have changed their structure. Check console for detection messages.
  • Extension not working: Make sure it's enabled and reload the page.
